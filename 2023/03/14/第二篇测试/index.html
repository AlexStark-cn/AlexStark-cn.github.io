如何搭建个人博客

  1. 安装git
    下载地址：https://git-scm.com/
  2. 安装node
    下载地址：https://nodejs.org/zh-cn/
  3. 安装博客框架：hexo
    下载地址：https://hexo.io/zh-cn/index.html
    1. 新建hexo文件夹
    2. 进入hexo文件夹，右键，选择Git Bash Here
    3. 依次输入hexo官网中提供的命令：
      • npm install hexo-cli -g这里输入代码
      • hexo init blog （创建了blog文件夹）
      • cd blog （打开blog文件夹）
      • npm install
      • hexo s （server的缩写）
    4. 复制出现的网址（注意不要用Ctrl+C复制，要用右键复制），再按下Ctrl+C终止命令。后把复制的网址粘贴到打开的浏览器中，出现如下图即可。

  4. 在当前blog文件夹中，单击右键，点击Git Bash Here。
  5. 在窗口中输入hexo new “文章标题”，按下回车，即可创建一篇文章。在source—>_posts文件夹中，即可看到这篇博客。
  6. 编辑该篇博客文本。
  7. 在git中输入hexo s运行博客。
  8. 打开博客，看到博客已经显示。

  9. 从hexo官网底部，点击探索主题，即可挑选免费主题。
