如何搭建个人博客

1. 配置环境

在指定文件夹下

  1. 安装git
    下载地址：https://git-scm.com/
  2. 安装node
    下载地址：https://nodejs.org/zh-cn/

    以上操作除了自己选选安装路径，可以无脑点下一步。

2. 安装hexo博客框架

Hexo地址：https://hexo.io/zh-cn/index.html
按照此网址中的提示部署hexo文件夹。

  1. 选择合适的位置新建hexo文件夹
  2. 进入hexo文件夹，右键，选择Git Bash Here（win11点击显示更多选项就能看见了）。
  3. 依次输入hexo官网中提供的命令：
    npm install hexo-cli -g
    hexo init blog #创建了blog文件夹
    cd blog #打开blog文件夹
    npm install
    hexo s #server的缩写
  4. 复制出现的网址

    注：

    1. 我的博客是编辑过的，因此在下图中显示了这么多东西。
    2. 注意不要用Ctrl+C复制，要用右键复制。
    3. 复制的是如图红框中的内容，一般是 http://localhost:4000。

3. 更改博客主题

  1. 打开hexo官方网站：https://hexo.io/zh-cn/index.html
  2. 下拉网页，点击“搜索主题”。
  3. 选择合适的主题进行预览，挑选主题。
  4. 点击所选主题的标题，即可进入该主题的github。
  5. 按照其github中的说明安装主题。有的主题有中文说明。

    大部分主题都是要你：

    1. 在Git Bash Here的界面中，用“git clone”命令安装主题
    2. 在hexo的根目录下的“_config.yml”文件中，修改theme字段的名字为你所下载主题的名字。
    3. 主题的框架内容要打开theme文件夹，选择你下载的主题对应的文件夹，修改该文件夹下的“_config.yml”文件。（注意：这个不是根目录下的“_config.yml”）

4. 在主题下创建blog

  1. 在当前blog文件夹中，单击右键，点击Git Bash Here。
  2. 在窗口中输入hexo new “文章标题”，按下回车，即可创建一篇文章。在source—>_posts文件夹中，即可看到这篇博客。
  3. 编辑该篇博客文本。
  4. 修改编好的文本的首部文本。除了默认的title、date、tags外，不同的主题有其自定的文本。如我这里用到了关于子页的category和时间线的selected。这些信息都在该主题的说明中。
  5. 在Git Bash Here中输入hexo g编译博客
  6. 输入hexo s运行博客。
  7. 打开 http://localhost:4000 看到博客已经显示。

5. 在github中部署博客

为了让自己的博客能够在互联网上被访问，需要把博客挂在到静态网页托管网站。如：国内的gitee（需要提前进行实名认证），国外的github（无需实名认证，但访问速度很慢，有时需要科学上网。）

  1. 打开github，登录你的账号。
  2. 新建一个仓库，点击如图所示
  3. 在repository name中，填写用户名.github.io

    然后点击最下方的Create repository。
  4. 回到Git Bash Here，依次输入以下代码，其中第一个名字随便写，邮箱换成自己的邮箱。
    git config --global user.name "名字"
    git config --global user.email abc@example.com
    ssh-keygen -t rsa -C "abc@example.com"
  5. 输入以下代码，即可获得密钥
    cat ~/.ssh/id_rsa.pub
  6. 点开github你创建的仓库，把密钥复制到如下界面，标题随便取。

    点击add SSH key添加密钥。
  7. 使用下面代码在Git Bash Here界面部署一个插件：
    npm install hexo-deployer-git --save
  8. 在根目录下的配置文件“_config.yml”中，在末尾补全信息

其中，repo后面的改成自己仓库的SSH，这个可以在建好的仓库中看到，如图：

9. 在命令行中执行hexo d即可把blog文件同步到github中。

6. 在github pages中部署自己的博客网页

  1. 打开部署好博客的仓库，点击Setting。
  2. 点击Pages。
  3. 如图中一样配置右边页面。
  4. 任何人都可以通过访问页面中给出的域名来打开你的博客。如我的博客地址：

7. 博客内容的更新同步

  1. 在本地博客文件中编辑/修改好博客。
  2. 点开Git Bash Here，依次输入：
    hexo g#编译
    hexo s#生成网页
    本地检查网页
    按Ctrl+C退出
    hexo d同步到github
    每次同步后需要几分钟的时间等待。
